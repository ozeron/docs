## Installation

Install the Sandbox SDK using your preferred package manager:

<CodeGroup>
```bash npm
npm install @usesandbox/sdk
```

```bash yarn
yarn add @usesandbox/sdk
```

```bash pnpm
pnpm add @usesandbox/sdk
```

```bash bun
bun add @usesandbox/sdk
```
</CodeGroup>

## Authentication

Set your API key as an environment variable:

```bash
export SANDBOX_API_KEY=your-api-key-here
```

Or pass it directly when initializing the client:

```typescript
import { Sandbox } from "@usesandbox/sdk";

const client = Sandbox({
  apiKey: "your-api-key-here"
});
```

## Basic Usage

### Create a Sandbox

Create a new sandbox with a Docker image:

```typescript
const sandbox = await client.sandboxes.create({
  image: "node:lts-alpine"
});
```

With custom timeout (defaults to 10 minutes):

```typescript
const sandbox = await client.sandboxes.create({
  image: "node:lts-alpine",
  timeoutMs: 600000 // 10 minutes
});
```

### Execute Commands

Execute commands using array syntax (direct execution):

```typescript
const result = await sandbox.exec(["echo", "Hello World"]);
console.log(result.stdout); // "Hello World\n"
console.log(result.exitCode); // 0
```

Or using string syntax (shell execution):

```typescript
const result = await sandbox.exec("npm install && npm test");
```

### Get Sandbox Info

Fetch current sandbox state:

```typescript
const info = await sandbox.info();
console.log(info.state); // "started"
console.log(info.name); // "sandbox-node-1"
```

### List Sandboxes

List all sandboxes in your organization:

```typescript
const sandboxes = await client.sandboxes.list();
for (const sb of sandboxes) {
  console.log(`${sb.id}: ${sb.data.state}`);
}
```

### Refresh State

Refresh sandbox data from the server:

```typescript
const updated = await sandbox.refresh();
console.log(updated.state);
```

### Delete Sandbox

Clean up when you're done:

```typescript
await sandbox.kill();
```

## Complete Example

```typescript
import { Sandbox } from "@usesandbox/sdk";

const client = Sandbox();

// Create sandbox
const sandbox = await client.sandboxes.create({
  image: "node:lts-alpine"
});

// Execute commands
const result = await sandbox.exec(["node", "--version"]);
console.log(result.stdout); // v20.x.x

// Clean up
await sandbox.kill();
```
